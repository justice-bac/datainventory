# Self-hosted Source Control in Azure

This repository contains code and documentation for setting up and managing self-hosted data inventory solution using Azure services. It includes guides, scripts, and configurations to help users deploy and maintain their own source control systems in the cloud.

The deployment workflow is documented for both first-time setup and later updates. Start with the installation guide, then use the usage guide for the deployment sequence.
